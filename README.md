# Finance Tracking System (Django Backend)

### Overview
A robust financial management system built with Django, featuring Role-Based Access Control (RBAC) and automated financial analytics.

### Roles & Permissions
- **Admin**: Full CRUD access to all financial records.
- **Analyst**: Access to financial summaries and filtering tools.
- **Viewer**: Read-only access to basic records.

### Setup Instructions
1. Clone the repo: `git clone <your-repo-link>`
2. Install dependencies: `pip install -r requirements.txt`
3. Run migrations: `python manage.py migrate`
4. Start server: `python manage.py runserver`
